# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in mcp-server-go-quality, please report it privately via [GitHub Security Advisories](https://github.com/afshinator/mcp-server-go-quality/security/advisories/new).

Do not open a public issue.

## Scope

This policy covers the mcp-server-go-quality MCP server itself. Vulnerabilities in the underlying tools it wraps (golangci-lint, govulncheck, nilaway) should be reported to their respective maintainers.

## Supported Versions

| Version | Supported |
|---|---|
| latest | ✅ |
| < latest | ❌ |
