2. ensureToolsInstalled() → sequential go install

3. chan collect → sort.Slice(file, line) → []Diagnostic
4. shared timeout context (configurable, default 5m)

Key Design Decisions

Does

  • Prefer pure functions (no hidden state, predictable output)
  • One file per tool parser
  • Location extraction for uniform agent navigation
  • Native raw output preserved for deep context
  • Pre-flight sequential install (no race conditions)
  • bufio.Scanner for NDJSON govulncheck output
  • go.work support (checks go.mod fallback)
  • Configurable timeout in .go-quality.yaml

Doesn't

  • No caching between calls (YAGNI for now)
  • No stager for slow tools (parallel covers it)
  • No separate gocyclo / gocognit binaries
  • No golangci-lint config duplication
